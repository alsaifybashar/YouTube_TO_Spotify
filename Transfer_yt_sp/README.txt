# Move YouTube Songs to Spotify

The project transfers song from Youtube to Spotify by extracting name of the song and the artist.

## Features

- Authenticate with Spotify and YouTube using OAuth 2.0.
- Retrieve all user-created playlists from Spotify.
- Search for songs on Spotify based on the tracks in the user's playlists.
- Create new playlists with matched songs on Spotify.

## Prerequisites

- Python 3.12
- Spotify Developer Account
- Google Cloud Console Account
- YouTube Data API v3 enabled
- chromedriver or geckodriver
